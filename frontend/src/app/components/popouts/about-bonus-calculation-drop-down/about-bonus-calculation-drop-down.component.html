How are bonuses calculated?

1. Bonuses for Social performance

  • Each social performance has a target value and an actual value. The difference between these values determines your bonus.
  • Formula:
    • If the actual value matches the target value → you earn 50 points.
    • If the actual value exceeds the target → you earn 100 points.
    • If the actual value is 1 below the target → you earn 20 points.
    • If the actual value is 2 below the target → you earn 10 points.
    • If the actual value is more than 2 below → the bonus is 0.

2. Bonuses for Orders

Each order is evaluated based on several criteria:

  • Order Priority:

    • Priority 5 → + 100 points.
    • Priority 4 → + 50 points.
    • Priority 3 → + 25 points.
    • Priority 1 or 2 → 0.

  • Client Rating:

    • Rating 1 → + 100 points.
    • Rating 2 → + 50 points.
    • Rating 3 → + 25 points.
    • Rating 4 or 5 → 0.

  • Quantity of sold items

  • Order Value:

    • Formula: order priority + client rating + items * price per unit * coefficient (0.1).
    • A portion of the order's value is added to your bonus.

3. Total Bonus

  • The total bonus is the sum of all bonuses for social goals and orders.
  • This total is recorded in your annual report and is available for review.

How It Works in Practice

  1. The system automatically gathers data:

    • Information about social goals.
    • Details about all orders for the year.

  2. Bonus Calculation:

    • Formulas described above are applied to calculate your bonuses.

  3. Result:

    • Your final bonus is recorded in your report for the year.
